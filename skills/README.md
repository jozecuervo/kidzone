# Skills

Repo-local agent skills live here so Kidzone guidance stays portable across agents and harnesses.

Keep each skill in its own folder with a `SKILL.md`. Avoid harness-specific metadata, hidden home-directory state, or tool-specific configuration unless the repo explicitly needs it.

Current local skills:

- `kidzone-new-game`: capture and review a new mini-project idea before building.
- `kidzone-game-iteration`: modify, verify, and stack follow-up work on an existing mini-project.
