# Astro Bot-inspired eye display

A local-only canvas experiment with seven friendly robot faces, pointer-tracking
dot-matrix eyes, and a shared animation scheduler. Pause with the on-screen
button or Space key. The display pauses in hidden tabs and presents a static
scene when the operating system requests reduced motion.

The robot design and name are recognizable fan inspiration from PlayStation's
Astro Bot character. This is an unofficial, non-commercial learning project;
Sony Interactive Entertainment and Team Asobi are not affiliated with it.

The robot shell, eye renderer, and animation code in this folder are original
project code. The page does not load third-party assets or make network
requests.
